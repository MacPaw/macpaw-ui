### Formatting

You able to format the date using format and dateFormat props.

- **formatter** - the function that gets selected value as parameter and should return formatted string value.
- **dateFormat** - the date formatting pattern that will be used to format the date. This prop is used only if ***formatter***  is not provided. You can use any of the [formatting tokens](https://date-fns.org/v2.0.0-alpha.27/docs/format) to format the date.

<DatePicker
  value={new Date()}
  label="Select date"
  placeholder="Select date"
  dateFormat="dd.MM.yyyy"
/>

```
<DatePicker
  value={new Date()}
  label="Select date"
  placeholder="Select date"
  dateFormat="dd.MM.yyyy"
/>
```

<DatePicker
  value={new Date()}
  label="Select date"

/>

```
<DatePicker
  value={new Date()}
```

export const formatDate = (date) => {
  const day = date.getDate();
  const month = date.getMonth() + 1;
  const year = date.getFullYear();
  return `Day: ${day}, Month: ${month}, Year: ${year}`;
};

<DatePicker
  value={new Date()}
  label="Select date"
  placeholder="Select date"
  formatter={formatDate}
/>
